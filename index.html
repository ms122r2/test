Rec Engine Standalone Demo

Make sure you have built the standalone bundle with npm run build:component RecEngine before loading this page.

Update the snippet below with your Rec Engine configuration. At a minimum, provide:

  • data-form-id — a Directus form ID configured for the Rec Engine flow
  • window.__LXS_COMPONENTS_BASE_URL__ (or data-api-base) — the absolute URL to your hosted /api/rec-engine routes
  • Optionally data-tenant-id, data-theme, and data-collect-contact
